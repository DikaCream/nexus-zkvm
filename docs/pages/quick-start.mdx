# Quick Start

<Steps>

### 1. Install the Nexus zkVM

First, install Rust: [bit.ly/start-rust](https://bit.ly/start-rust).

Next, install the RISC-V target: `rustup target add riscv32i-unknown-none-elf`.

Then, install the Nexus zkVM:

```shell
cargo install --git https://github.com/nexus-xyz/nexus-zkvm nexus-tools
```

Verify the installation:

```shell
cargo nexus --help
```

This should print all the available commands.

### 2. Create a new Nexus project

```shell
cargo nexus new nexus-project
```

And change directory to the new project:

```shell
cd nexus-project
```

This will create a new Rust project with the following structure:

```shell
./nexus-project
├── Cargo.lock
├── Cargo.toml
└── src
    └── main.rs
```

And an example zkVM Rust program in `./src/main.rs`:

```rust
#![no_std]
#![no_main]

use nexus_rt::write_log;

#[nexus_rt::main]
fn main() {
    write_log("Hello, World!\n");
}
```

### 3. Run your program

```shell
cargo nexus run
```

You should see the program print:

```shell
"Hello, World!"
```

### 3. Prove your program

Generate a zero-knowledge proof for your Rust program using the Nexus zkVM.

```shell
cargo nexus prove -r
```

> Note: we use the `-r` flag to prove the program in Rust *release* mode, which is significantly faster.

This will generate a proof, and store it in `./nexus-proof.json`.

### 4. Verify your proof

Finally, load and verify the proof:

```shell
cargo nexus verify
```

You should see the program print:

```shell
INFO nexus-tools: Proof is valid
```

</Steps>
