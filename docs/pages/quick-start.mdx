# Quick Start

<Steps>

### 1. Install the Nexus zkVM

First install Rust: [bit.ly/start-rust](https://bit.ly/start-rust).

Then build and install the Nexus zkVM:

```shell
git clone https://github.com/nexus-xyz/nexus-zkvm
cd nexus-zkvm
cargo install --path .
```

### 2. Create a new Nexus project

```shell
cargo nexus new nexus-project
```

This will create a directory with the following structure:

```shell
./nexus-project
├── Cargo.lock
├── Cargo.toml
└── src
    └── main.rs
```

### 3. Prove your program

Generate a zero-knowledge proof for your Rust program using the Nexus zkVM:

```shell
cargo nexus prove
```

### 4. Verify your proof

```shell
cargo nexus verify
```

</Steps>

It's that simple. Next, learn how to connect to the Nexus Network to produce proofs at production-grade scale.
